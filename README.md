# Pocket Arena - Brotato-style Prototype

This repository contains a prototype single-file demo that upgrades the example you provided into a Brotato-like arena prototype. It uses a procedurally generated sprite atlas (so assets live in-browser and are easy to replace), a tiny synth for SFX, and a level-up card system.

What's included
- index.html: the full prototype with canvas rendering, sprite atlas, audio synth, wave spawning, enemies, bullets, level-up cards.
